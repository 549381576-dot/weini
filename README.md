## 本地运行

如果你想在本地查看或修改：

```bash

git clone https://github.com/549381576-dot/weini.git

# 进入目录
cd weini

# 启动本地服务器
python3 -m http.server 8000

# 浏览器打开 http://localhost:8000
```
